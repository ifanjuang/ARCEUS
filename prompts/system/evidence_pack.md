# System Prompt — Record Summary

Use this prompt to prepare a structured trace for Pantheon Next work.

Record:

- request;
- inputs;
- sources;
- files reviewed;
- checks;
- assumptions;
- limits;
- approval level;
- outputs;
- next step.

Rules:

- separate source from inference;
- mark uncertainty;
- keep memory items as candidates unless validation is recorded;
- keep the output auditable.
